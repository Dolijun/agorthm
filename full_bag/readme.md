## 问题描述
利用动态规划算法实现广义背包问题
广义背包问题的描述如下：给定载重量为M的背包和n种物品，每种物品有一定的重量和价值，现在需要设计算法，在不超过背包载重量的前提下，巧妙选择物品，使得装入背包的物品的总价值最大化。规则是，每种物品均可装入背包多次或不装入（但不能仅装入物品的一部分）。

需要给出最优值和最优解，即装入物品的最大总价值，以及具体每种物品装入几个。
